# ACT - Making Sense of Gender Equality Survey Data

This document provides hands-on guidance for carrying out statistical analysis of survey results generated by the Gender Equality Audit and Monitoring (GEAM) tool. 

The ACT GEAM tool is a modular questionnaire framework which captures the perceptions of employees regarding key dimensions of gender equality within an organizational unit. It has been developed in the framework of the ACT project. See [https://www.act-on-gender.eu](https://www.act-on-gender.eu) for details. 

To use the GEAM survey instrument please visit [https://geam.act-on-gender.eu](https://geam.act-on-gender.eu) and apply for an account.
